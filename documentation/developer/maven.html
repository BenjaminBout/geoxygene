GeOxygene Library for Maven Developer

Project settings

This tutorial is for developers who wants use GeOxygene library in a maven project.

Open the pom.xml file and configure :

  • Repositories to download jars :

    <repositories>
	<repository>
		<id>cogit-snapshots</id>
		<name>Cogit Snapshots Repository</name>
		<url>https://forge-cogit.ign.fr/nexus/content/repositories/snapshots/</url>
	</repository>
	<repository>
		<id>cogit-releases</id>
		<name>Cogit Releases Repository</name>
		<url>https://forge-cogit.ign.fr/nexus/content/repositories/releases/</url>
	</repository>
</repositories>

  • the dependencies like this :

    <properties>
	<project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    <geoxygene.version>1.8-SNAPSHOT</geoxygene.version>
</properties>
	
<dependencies>
    <dependency>
        <groupId>fr.ign.cogit</groupId>
        <artifactId>geoxygene-XXX</artifactId>
        <version>${geoxygene.version}</version>
    </dependency>
</dependencies>
