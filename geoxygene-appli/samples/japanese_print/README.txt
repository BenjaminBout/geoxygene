############### HOWTO ###############
1) Create a backup of the file geoxygene-appli/geoxygene-config.xml if it exists
2) Edit the file geoxygene-config.xml
3) Copy the geoxygene-config.xml to geoxygene-appli/
4) Run Geoxygene
