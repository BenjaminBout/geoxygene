############### INFO ###############
This README is written for the SLDs aquarelle1.xml and aquarelle2.xml

############### HOWTO ###############
1) Create a backup of the file geoxygene-appli/geoxygene-config.xml if it exists
2) Edit the file geoxygene-config_1.xml or geoxygene-config_2.xml 
3) Copy the file geoxygene-config_1.xml  or geoxygene-config_2.xmlto geoxygene-appli/ and rename it geoxygene-config.xml
4) Run Geoxygene
