

Geometrie 
###########

Introduction
**************

Dans GeOxygene la dimension spatiale des objets géographiques repose sur la large gamme de primitives géométriques 
et topologiques spécifiée par la norme ISO 19107.


Exemple
*********

Pb : "Déplacer" les Etats-Unis afin d'avoir une enveloppe qui ne soit pas celle de la mappemonde .... 

s'il y a des personnes qui veulent réaliser une carte à l'échelle du monde via GeOxygène : 
c'est une astuce qui permet du coup de centrer facilement une carte sur une longitude particulière
